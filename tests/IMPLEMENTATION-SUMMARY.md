## 运行指南

### 快速开始

```bash
# 1. 进入测试目录
cd tests

# 2. 安装依赖
npm install
npm run test:e2e:install

# 3. 启动测试环境
docker-compose -f ../docker-compose.test.yml up -d

# 4. 生成 Mock 数据
npm run generate-mock-data

# 5. 运行测试
npm run test:e2e

# 或使用一键脚本
./scripts/run-e2e-tests.sh all true
```

### 高级用法

```bash
# 运行特定测试套件
npx playwright test import-flow.spec.ts
npx playwright test permission-flow.spec.ts

# 以 headed 模式运行
npm run test:e2e:headed

# UI 模式调试
npm run test:e2e:ui

# 查看测试报告
npm run test:e2e:report
```
